# Datasets

Data files are not included in this repository.

## 1. Fake/True News Dataset
Download from Kaggle:
https://www.kaggle.com/datasets/clmentbisaillon/fake-and-real-news-dataset

Place `Fake.csv` and `True.csv` in this folder.

## 2. LIAR Dataset
Official source (Hugging Face — hosted by original authors):
https://huggingface.co/datasets/ucsbnlp/liar

Also available on Kaggle:
https://www.kaggle.com/datasets/doanquanvietnamca/liar-dataset

Place `train.tsv`, `valid.tsv`, `test.tsv` inside a `liar_dataset/` subfolder.
